Data Analytics

in Plain English

What is Data Analytics?

Analytics are off by default. If you turn it on, your app sends us a few anonymous numbers each day — a digital headcount. We can see that someone sent 10 messages today, but we have no idea if that someone was you. No names, no messages, no photos. Just numbers.

Why does it help?

These numbers help us fix bugs, plan for busy periods, and make the app run better for everyone. We never sell or share this data. Ever.

Who does it help?

Honestly, it helps everyone who uses Pals, including your Pals. But you should only turn this on if you trust the people who make this app.

Where is the data sent?

The data is sent to a secure Pals server, where it is combined and analyzed with data from other Pals. The data cannot be tracked back to you.

What data does Pals collect?

The only data sent are numbers:

  • How often the app is opened
  • How many messages are sent or received
  • Any errors, like a message failing to send or a connection problem

We collect the minimum amount of data possible, and nothing else.

How can I view the data you've collected?

You can, it's viewable [here]. Once analytics are sent, the file is reset. You can view it anytime. (You'll know if the file was sent recently if the file doesn't contain any data.)
